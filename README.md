## 💡 فكرة المشروع

**NoorifyBot** هو بوت تيليجرام بسيط يهدف إلى تذكير المستخدم بالأذكار والأدعية والآيات القصيرة بشكل دوري داخل:
* **الخاص** (المحادثات الفردية)
* **المجموعات**
* **القنوات**

> ﴿ وَذَكِّرْ فَإِنَّ الذِّكْرَىٰ تَنْفَعُ الْمُؤْمِنِينَ ﴾

---

## ✨ الميزات

* 📿 **أذكار وأدعية عشوائية:** إرسال محتوى متجدد في كل مرة.
* ⏰ **تذكير دوري:** فترات زمنية مرنة تبدأ من (30 دقيقة ← يومي).
* 🔁 **منع التكرار:** خوارزمية ذكية لمنع تكرار الأذكار المتتالية.
* 💬 **دعم الشمولية:** متوافق بالكامل مع الخاص، المجموعات، والقنوات.
* 🔘 **واجهة أزرار بسيطة:** تجربة مستخدم سلسة وسهلة التفاعل.
* ⚙️ **تحكم كامل:** إمكانية تشغيل وإيقاف التذكير بكل سهولة.

## ⚙️ التشغيل والتثبيت

### 📥 خطوات التشغيل
قم بتشغيل الأوامر التالية في الطرفية (Terminal):

```bash
# استنساخ المستودع
git clone [https://github.com/yourusername/noorifybot](https://github.com/yourusername/noorifybot)

# الدخول إلى مجلد المشروع
cd noorifybot

# تثبيت المكتبات المطلوبة
pip install -r requirements.txt

# تشغيل البوت
python bot.py

```

### 🔑 الإعداد (Configuration)

قم بتعيين توكن البوت الخاص بك في ملف الإعدادات:

```python
TOKEN = "YOUR_BOT_TOKEN"

```

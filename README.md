## 💡 فكرة المشروع

**NoorifyBot** هو بوت تيليجرام بسيط يهدف إلى تذكير المستخدم بالأذكار والأدعية والآيات القصيرة بشكل دوري داخل:

- الخاص
- المجموعات
- القنوات

﴿ وَذَكِّرْ فَإِنَّ الذِّكْرَىٰ تَنْفَعُ الْمُؤْمِنِينَ ﴾

---

## ✨ الميزات

- 📿 أذكار وأدعية عشوائية
- ⏰ تذكير دوري (30 دقيقة → يومي)
- 🔁 منع تكرار الأذكار
- 💬 دعم الخاص والمجموعات والقنوات
- 🔘 واجهة أزرار بسيطة وسهلة
- ⚙️ تشغيل وإيقاف التذكير
